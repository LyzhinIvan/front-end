NPM — NodeJS Package Manager

https://www.npmjs.com

NPM

NPM — конскольная утилита, которая предоставляет доступ к разным пакетам сторонних разработчиков.

Поиск пакета

npm search name

Просмотр пакета

npm view name

Установка пакета

npm install name

Типы зависимостей

  • Основные (dependencies)
  • Для разработки (devDependencies)

Пути установки пакета

  • Локально
  • Глобально

Установка пакета глобально

npm install -g name

Установка определенной версии пакета

npm install name@1.0.2

Обновление пакета

npm update name

Удаление пакета

npm uninstall name

npm remove name

Создание пакета

npm init

Установка пакета

C сохранением зависимости в package.json

npm install name --save

Установка пакета

C сохранением dev-зависимости в package.json

npm install name --save-dev

Восстановление пакетов

npm install
